# 大学默写

一个大学语文背诵默写练习网页应用。

## 功能特点

- **填句成章**：给出部分原文，拖动句子填空
- **全文排序**：打乱句子顺序，拖动排列正确顺序
- **默写填空**：传统打字填空，支持自定义挖空选择
- **背诵模式**：自我评估背诵效果，检查答案后可修改

## 难度设置

- 简单：20% 挖空
- 适中：40% 挖空（默认）
- 困难：60% 挖空

## 使用方式

1. 在左侧选择篇目
2. 在右侧选择练习模式
3. 根据提示完成练习
4. 点击检查查看结果
5. 点击再练一次重新开始

## 技术栈

- HTML5
- CSS3
- JavaScript

## 运行方式

```bash
# 使用本地服务器运行
python -m http.server 8080
# 或
npx http-server -p 8080
```

访问 http://localhost:8080/
